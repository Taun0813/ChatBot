Tổng quan hệ thống AI Agent
AI Agent System là hệ thống trí tuệ nhân tạo thông minh được thiết kế cho lĩnh vực thương mại điện tử, đặc biệt là tư vấn và tìm kiếm sản phẩm điện thoại.
Kiến trúc chính
1. Hybrid Orchestrator (Bộ điều phối lai)
    Kết hợp 2 phương pháp routing:
    Rule-based: Nhanh, dựa trên quy tắc cố định
    ML-based: Thông minh, học từ ngữ cảnh
    Decision Fusion Engine: Kết hợp quyết định từ cả 2 phương pháp
    Độ chính xác: 85-95%
2. RAG System (Retrieval-Augmented Generation)
    Sử dụng Pinecone vector database
    Tìm kiếm ngữ nghĩa trong dataset 27,000+ sản phẩm điện thoại thực tế
    Hỗ trợ các thương hiệu: OnePlus, Samsung, Apple, Xiaomi, Motorola, Realme, Nothing
    Embedding model: llama-text-embed-v2
3. Multi-Model Support
    Hỗ trợ nhiều LLM: Gemini, Groq, Ollama, OpenAI, Claude
    Model switching strategy: round-robin
    Fallback mechanism khi model không khả dụng
4. Personalization System
    Profile Manager: Quản lý hồ sơ người dùng
    Recommender: Gợi ý sản phẩm dựa trên sở thích
    RL Feedback: Học từ phản hồi người dùng
    Phân tích hành vi và lịch sử tìm kiếm
5. API Integration
    Tích hợp với Spring Boot microservices
    Hỗ trợ: Order service, Payment service, Warranty service, Product service
    Mock data fallback khi service không khả dụng
6. Caching System
    Redis cache cho production
    Memory cache cho development
    Cache responses và embeddings
    TTL và invalidation thông minh
7. Monitoring & Observability
    Metrics collection: API latency, success rate, error rate
    Health checks: System resources, application health
    Request tracing: OpenTelemetry/Jaeger integration
    Real-time dashboard
Luồng xử lý
    Nhận request từ người dùng
    Hybrid Orchestrator phân tích và quyết định routing
    RAG Agent tìm kiếm sản phẩm phù hợp
    Personalization điều chỉnh kết quả theo sở thích người dùng
    API Agent xử lý các yêu cầu về đơn hàng, thanh toán, bảo hành
    Conversation Agent tạo phản hồi tự nhiên
    Cache lưu trữ kết quả để tăng tốc
    Monitoring ghi lại metrics và tracing
Dataset thực tế
    27,000+ sản phẩm điện thoại từ các thương hiệu lớn
    Thông số chi tiết: CPU, RAM, ROM, camera, pin, màn hình, 5G, NFC
    Giá cả thực tế: Từ 19,989 VNĐ đến hàng triệu VNĐ
    Rating và reviews từ người dùng thực tế
Tính năng nổi bật
    Tìm kiếm thông minh: Hiểu ngữ cảnh và ý định người dùng
    Cá nhân hóa: Học từ hành vi và đưa ra gợi ý phù hợp
    Tích hợp API: Kết nối với hệ thống quản lý đơn hàng, thanh toán
    Multi-model: Sử dụng nhiều LLM để tối ưu hiệu suất
    Caching: Tăng tốc phản hồi với hệ thống cache thông minh
    Monitoring: Theo dõi hiệu suất real-time
Hệ thống được thiết kế modular, có thể mở rộng và tùy chỉnh theo nhu cầu cụ thể của từng doanh nghiệp.
